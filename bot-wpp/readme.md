# Bot de Direcionamento WhatsApp

Bot automático desenvolvido em Python utilizando a WhatsApp Cloud API.

## Funcionalidades
- Menu inicial de setores
- Resposta automática por opção
- Integração oficial com a Meta

## Requisitos
- Conta Meta Business
- WhatsApp Cloud API habilitada
- Servidor com Python 3.10+
- HTTPS ativo

## Execução
```bash
pip install -r requirements.txt
python app.py
